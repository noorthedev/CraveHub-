**🍔 CraveHub – Modern Restaurant Website**

CraveHub is a fully responsive restaurant website designed to deliver a modern and engaging user experience. It allows users to explore the menu, add items to the cart or wishlist, reserve tables, contact the restaurant, and enjoy a seamless browsing experience across all devices.

✨ Features
🍕 Modern & Responsive UI
🍔 Interactive Food Menu
🛒 Shopping Cart
❤️ Wishlist
📍 Table Reservation
📧 Contact Form with EmailJS
🌙 Dark & Light Mode
🔍 Food Search
⭐ Customer Testimonials
📱 Mobile-Friendly Design
⬆️ Scroll-to-Top Button
⚡ Clean & Optimized Code
🛠️ Technologies Used
HTML5
CSS3
JavaScript (ES6)
EmailJS
AOS (Animate On Scroll)
Ionicons
🎯 Purpose

This project was built to enhance frontend development skills by implementing real-world restaurant website functionality with a clean, responsive, and interactive design.
